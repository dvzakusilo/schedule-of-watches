# visual-code-review

 Simple library for visual fixing of the revealed defects of imposition.

## Installation

`npm i schedule-of-watches `

## Usage

```html
<div id="app">
    <Calendar :ar-names="['Иванов И.И.','Петров П.П.','Сидоров С.С.']"></Calendar>
</div>
```
